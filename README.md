FitLog: Gym Data Pipeline

FitLog is a Python-based data pipeline that processes fitness tracking data from raw CSV files into validated, structured insights and reports.


## project structure
fitlog/
├── data/
│   ├── workouts.csv
│   ├── body_metrics.csv
│   └── exercise_catalog.csv
├── fitlog/
│   ├── __init__.py        
│   ├── models.py           
│   ├── parser.py            
│   ├── validators.py        
│   ├── analytics.py         
│   └── reports.py           
├── output/                  
├── explore.py
└── README.md
